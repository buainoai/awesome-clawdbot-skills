# 关于本项目

## 项目简介

本项目是 [VoltAgent/awesome-clawdbot-skills](https://github.com/VoltAgent/awesome-clawdbot-skills) 的中文翻译版本，旨在为中文用户提供更便捷的 Clawdbot 技能查找和使用体验。

## 什么是 Clawdbot？

**Clawdbot** 是一款本地运行的 AI 助手，直接在您的设备上操作。它通过 **Skills（技能）** 来扩展功能，使其能够：

- 与外部服务交互
- 自动化工作流程
- 执行专门任务

## 如何使用

### 1. 浏览技能
查看 `README.md` 中的分类列表，找到您需要的技能。

### 2. 安装技能

#### 方法一：使用 ClawdHub CLI（推荐）
```bash
npx clawdhub@latest install <skill-slug>
```

#### 方法二：手动安装
将技能文件夹复制到以下任一位置：

| 位置 | 路径 |
|------|------|
| 全局 | `~/.clawdbot/skills/` |
| 工作区 | `<project>/skills/` |

**优先级**：工作区 > 本地 > 内置

### 3. 使用技能
安装后，Clawdbot 会自动识别并在需要时使用相应的技能。
